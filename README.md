# 📌 프로젝트 실행 방법

1. Clone the repo

```javascript
$ git clone https://github.com/wanted-pre-onboarding-team-9/pre-onboarding-10th-4-9.git
```

2. Install Yarn packages

```javascript
$ yarn install
```

3. Getting Started

```javascript
$ yarn start
```
